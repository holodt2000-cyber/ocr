# 🎉 Поддержка PDF успешно добавлена!

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Установите Poppler

**Windows:**
1. Откройте `POPPLER_INSTALL.md`
2. Следуйте инструкциям
3. Перезапустите командную строку

**macOS:**
```bash
brew install poppler
```

**Linux:**
```bash
sudo apt-get install poppler-utils
```

### Шаг 2: Проверьте установку

**Windows:**
```bash
TEST_PDF_NOW.bat
```

**Linux/macOS:**
```bash
./test_pdf_now.sh
```

Или вручную:
```bash
python scripts/test_pdf.py
```

### Шаг 3: Начните использовать!

**Веб-интерфейс:**
```bash
# Windows
START.bat

# Linux/macOS
python3 web_app.py
```

Откройте http://localhost:5000 и загрузите PDF!

**Командная строка:**
```bash
# Обработать весь PDF
python main.py document.pdf

# Обработать страницу 3
python main.py document.pdf 3

# Пакетная обработка
python scripts/batch_pdf_ocr.py ./pdfs ./output
```

## ⚠️ Устранение проблем

### "PDF support not available"
**Решение:**
1. Установите Poppler (см. `POPPLER_INSTALL.md`)
2. Перезапустите терминал
3. Запустите `python scripts/test_pdf.py`

### "Failed to convert PDF"
**Возможные причины:**
- PDF поврежден
- PDF защищен паролем
- Недостаточно памяти

**Решение:**
- Проверьте PDF в другой программе
- Попробуйте другой PDF файл
- Обрабатывайте по одной странице

### Медленная обработка
**Решение:**
- Обрабатывайте страницы по отдельности
- Уменьшите DPI в `utils/pdf_processor.py` (с 300 до 200)

## 💡 Советы

1. **Качество OCR** зависит от качества PDF
   - Используйте PDF с разрешением 300+ DPI
   - Четкий, контрастный текст дает лучшие результаты

2. **Производительность**
   - Большие PDF обрабатывайте по страницам
   - Используйте пакетную обработку для множества файлов

3. **Редактирование**
   - Всегда проверяйте результаты OCR
   - Используйте веб-интерфейс для удобного редактирования
