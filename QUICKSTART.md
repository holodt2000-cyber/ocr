# Быстрый старт

## Установка

1. Установите Python 3.7 или новее
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Установите Tesseract OCR:
   - Windows: См. `POPPLER_INSTALL.md`
   - macOS: `brew install tesseract tesseract-lang`
   - Linux: `sudo apt-get install tesseract-ocr tesseract-ocr-rus tesseract-ocr-eng`

## Запуск

### Windows
Просто запустите:
```
START.bat
```

### Linux/macOS
```bash
python web_app.py
```

Откройте браузер: http://localhost:5000

## Использование

1. **Загрузите изображение** - нажмите "Открыть изображение"
2. **Распознайте текст** - нажмите "Распознать текст"
3. **Редактируйте** - кликните на текст для выбора, перетаскивайте мышью
4. **Добавьте метки** - нажмите "Добавить метку" и кликните на изображение
5. **Сохраните** - нажмите "Сохранить"

## Возможности

- ✅ Распознавание текста (OCR) на русском и английском
- ✅ Интерактивное редактирование текста
- ✅ Перетаскивание текстовых блоков
- ✅ Добавление меток кликом
- ✅ Удаление ненужных элементов
- ✅ Сохранение результата

## Решение проблем

### Tesseract не найден
Убедитесь, что Tesseract установлен и добавлен в PATH.
См. `POPPLER_INSTALL.md` для подробных инструкций.

### Ошибка при загрузке изображения
Поддерживаемые форматы: PNG, JPG, JPEG, BMP, TIFF
Максимальный размер: 16 МБ

### Плохое качество распознавания
- Используйте изображения с хорошим разрешением
- Убедитесь, что текст четкий и контрастный
- Проверьте, что установлены языковые файлы (eng.traineddata, rus.traineddata)
