# Быстрый старт

## Установка

1. Установите Python 3.7 или новее
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Установите Tesseract OCR:
   - Windows: См. `POPPLER_INSTALL.md`
   - macOS: `brew install tesseract tesseract-lang`
   - Linux: `sudo apt-get install tesseract-ocr tesseract-ocr-rus tesseract-ocr-eng`

4. (Опционально) Для работы с PDF установите Poppler:
   - Windows: См. `POPPLER_INSTALL.md`
   - macOS: `brew install poppler`
   - Linux: `sudo apt-get install poppler-utils`

## Запуск

### Windows
Просто запустите:
```
START.bat
```

### Linux/macOS
```bash
python web_app.py
```

Откройте браузер: http://localhost:5000

## Использование

### Веб-интерфейс

1. **Загрузите файл** - нажмите "Открыть файл" (поддерживаются изображения и PDF)
2. **Для PDF** - используйте кнопки навигации для переключения между страницами
3. **Распознайте текст** - нажмите "Распознать текст"
4. **Редактируйте** - кликните на текст для выбора, перетаскивайте мышью
5. **Добавьте метки** - нажмите "Добавить метку" и кликните на изображение
6. **Сохраните** - нажмите "Сохранить"

### Командная строка

```bash
# Обработать изображение
python main.py image.png

# Обработать PDF (все страницы)
python main.py document.pdf

# Обработать конкретную страницу PDF
python main.py document.pdf 3

# Пакетная обработка PDF
python scripts/batch_pdf_ocr.py ./pdfs ./output
```

## Решение проблем

### Tesseract не найден
Убедитесь, что Tesseract установлен и добавлен в PATH.
См. `POPPLER_INSTALL.md` для подробных инструкций.

### Ошибка при загрузке файла
Поддерживаемые форматы: PNG, JPG, JPEG, BMP, TIFF, PDF
Максимальный размер: 16 МБ

### PDF не открывается
- Убедитесь, что установлен Poppler (см. `POPPLER_INSTALL.md`)
- Проверьте, что PDF не защищен паролем
- Попробуйте открыть PDF в другой программе для проверки

### Плохое качество распознавания
- Используйте изображения с хорошим разрешением
- Убедитесь, что текст четкий и контрастный
- Проверьте, что установлены языковые файлы (eng.traineddata, rus.traineddata)
