# Деплой на Render

## Подготовка

1. Убедитесь, что все изменения закоммичены в Git
2. Создайте аккаунт на [Render.com](https://render.com)

## Настройка для Render

### 1. Создайте файл `render.yaml` (уже создан)

### 2. Переменные окружения

В настройках Render добавьте:
- `PYTHON_VERSION`: `3.11`

### 3. Важные моменты

- EasyOCR будет использовать CPU режим (gpu=False) для совместимости
- Первый запуск может занять 5-10 минут (загрузка моделей)
- Модели EasyOCR кэшируются автоматически

## Деплой

1. Подключите GitHub репозиторий к Render
2. Выберите "Web Service"
3. Настройки:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn web_app:app`
   - **Environment**: Python 3

4. Нажмите "Create Web Service"

## Оптимизация

Для ускорения работы на Render:
- Используется EasyOCR вместо Tesseract (не требует системных зависимостей)
- CPU режим для совместимости с бесплатным тарифом
- Модели загружаются один раз при старте

## Ограничения бесплатного тарифа

- 512 MB RAM (может быть недостаточно для больших PDF)
- Сервис засыпает после 15 минут неактивности
- Первый запуск после сна занимает ~30 секунд

## Рекомендации

Для production использования рекомендуется:
- Платный тариф с большим объемом RAM
- Включение GPU для ускорения (платные тарифы)
- Настройка кэширования моделей
