## 🚀 Использование

### Запуск
```bash
START.bat
```

Или:
```bash
python web_app.py
```

Откройте: http://localhost:5000

### Основные функции

1. **Upload Document** - загрузка изображения или PDF
2. **Run OCR** - распознавание текста
3. **White Background** - режим белого фона
4. **Reset Zoom** - сброс масштаба
5. **Save** - сохранение результата

### PDF навигация
- **Previous** - предыдущая страница
- **Next** - следующая страница
- **Page indicator** - текущая страница / всего страниц

### Редактирование
- **Клик** на текст для выбора
- **Перетаскивание** для перемещения
- **Edit Selected Text** - редактирование в панели
- **Update** - сохранение изменений
- **Delete** - удаление элемента

## 🎨 Дизайн-система

### Типографика
- **Заголовки:** Manrope (800, 700, 600)
- **Текст:** Inter (600, 500, 400)
- **Размеры:** 5xl, 2xl, xl, lg, base, sm, xs

### Компоненты
- **Кнопки:** Градиенты, solid, outline
- **Карточки:** Rounded-xl с тенями
- **Inputs:** Rounded-lg с focus states
- **Badges:** Rounded-full с цветными фонами

### Spacing
- **Padding:** 2, 3, 4, 6, 8, 12
- **Gap:** 2, 3, 4, 6, 8
- **Margin:** 2, 4, 8, 12
