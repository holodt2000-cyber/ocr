┌──────────────────────────────────────────────────────────────────────────────┐
│  ИСПОЛЬЗОВАНИЕ                                                               │
└──────────────────────────────────────────────────────────────────────────────┘

1. ЗАГРУЗКА:
   → Нажмите "Upload Document"
   → Выберите изображение или PDF
   → Документ откроется автоматически

2. OCR:
   → Нажмите "Run OCR"
   → Текст распознается и появится справа
   → Блоки текста появятся на изображении

3. РЕДАКТИРОВАНИЕ:
   → Кликните на текстовый блок
   → Редактируйте в панели справа
   → Нажмите "Update"

4. ПЕРЕМЕЩЕНИЕ:
   → Перетаскивайте блоки мышью
   → Позиция сохраняется автоматически

5. МАСШТАБИРОВАНИЕ:
   → Ctrl + колесо вверх → увеличение
   → Ctrl + колесо вниз → уменьшение
   → "Reset Zoom" → подгонка под экран

6. PDF НАВИГАЦИЯ:
   → "Previous" → предыдущая страница
   → "Next" → следующая страница
   → Индикатор: "Page 3 / 10"

7. БЕЛЫЙ ФОН:
   → "White Background" → скрыть изображение
   → Видны только текстовые блоки
   → Удобно для копирования

8. СОХРАНЕНИЕ:
   → "Save" → скачать результат
   → Формат: PNG с текстом
