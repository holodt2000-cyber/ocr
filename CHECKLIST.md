# Чеклист проверки

## Перед запуском

- [ ] Python 3.7+ установлен
- [ ] Установлены зависимости: `pip install -r requirements.txt`
- [ ] Tesseract OCR установлен
- [ ] Языковые файлы (eng.traineddata, rus.traineddata) в папке tessdata

## Проверка работоспособности

### 1. Запуск приложения
```bash
python web_app.py
```
Ожидаемый результат: Сервер запустился на http://localhost:5000

### 2. Открытие в браузере
Откройте: http://localhost:5000
Ожидаемый результат: Загрузилась страница с интерфейсом

### 3. Загрузка изображения
- Нажмите "Открыть изображение"
- Выберите изображение (PNG, JPG, JPEG, BMP, TIFF)
Ожидаемый результат: Изображение отображается

### 4. OCR
- Нажмите "Распознать текст"
Ожидаемый результат: Текст распознан и отображается в списке справа

### 5. Редактирование
- Кликните на текстовый блок
- Измените текст в поле редактирования
- Нажмите "Обновить"
Ожидаемый результат: Текст обновился

### 6. Перетаскивание
- Зажмите текстовый блок мышью
- Перетащите в другое место
Ожидаемый результат: Блок переместился

### 7. Добавление метки
- Нажмите "Добавить метку"
- Кликните на изображение
- Введите текст
Ожидаемый результат: Новая метка добавлена

### 8. Удаление
- Выберите текстовый блок
- Нажмите "Удалить"
Ожидаемый результат: Блок удален

### 9. Сохранение
- Нажмите "Сохранить"
Ожидаемый результат: Файл скачался

## Возможные проблемы

### Tesseract не найден
**Решение:** Установите Tesseract и добавьте в PATH
См. POPPLER_INSTALL.md

### Ошибка импорта модулей
**Решение:** `pip install -r requirements.txt`

### Порт 5000 занят
**Решение:** Измените порт в web_app.py (строка с app.run)

### Плохое качество OCR
**Решение:** 
- Используйте четкие изображения
- Проверьте наличие языковых файлов
- Увеличьте разрешение изображения
