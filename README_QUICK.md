# Archivist OCR - Quick Start

## 🚀 Запуск

```bash
START.bat
```

Откройте: http://localhost:5000

## ✨ Возможности

- 📄 Загрузка изображений и PDF
- 🔍 OCR (Русский + Английский)
- ✏️ Редактирование текста
- 🖱️ Перетаскивание блоков
- 🔎 Масштабирование (Ctrl + колесо)
- 📑 PDF навигация
- ⚪ Белый фон режим
- 💾 Сохранение результата
