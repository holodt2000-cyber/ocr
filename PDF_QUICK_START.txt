┌──────────────────────────────────────────────────────────────────────────────┐
│ БЫСТРАЯ УСТАНОВКА                                                            │
└──────────────────────────────────────────────────────────────────────────────┘

1. Установите Poppler:

   Windows:  См. POPPLER_INSTALL.md
   macOS:    brew install poppler
   Linux:    sudo apt-get install poppler-utils

2. Проверьте установку:

   python scripts/test_pdf.py

3. Готово! Теперь можете работать с PDF

┌──────────────────────────────────────────────────────────────────────────────┐
│ ИСПОЛЬЗОВАНИЕ                                                                │
└──────────────────────────────────────────────────────────────────────────────┘

▶ ВЕБ-ИНТЕРФЕЙС:
  
  1. Запустите: START.bat (или python web_app.py)
  2. Откройте: http://localhost:5000
  3. Загрузите PDF файл
  4. Используйте кнопки "◀ Назад" и "Вперед ▶"
  5. Запустите OCR на нужной странице

▶ КОМАНДНАЯ СТРОКА:

  # Обработать весь PDF
  python main.py document.pdf

  # Обработать страницу 3
  python main.py document.pdf 3

  # Пакетная обработка
  python scripts/batch_pdf_ocr.py ./pdfs ./output

❌ "PDF support not available"
   → Установите Poppler (см. POPPLER_INSTALL.md)
   → Перезапустите терминал
   → Запустите: python scripts/test_pdf.py

❌ "Failed to convert PDF"
   → Проверьте, что PDF не поврежден
   → Убедитесь, что PDF не защищен паролем

❌ Медленная обработка
   → Обрабатывайте по одной странице
   → Уменьшите DPI в utils/pdf_processor.py
