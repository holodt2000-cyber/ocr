# Текущее состояние проекта OCR Web Editor

Дата: 2026-04-12

## ✅ Что работает:

### Backend (100% готов)
- Многостраничная поддержка PDF
- Каждая страница хранит свои метки и состояние фона
- Poppler интегрирован (путь: C:\Program Files\poppler\Library\bin)
- Endpoints готовы:
  - `/upload` - загрузка и конвертация PDF в страницы
  - `/ocr` - распознавание текста на текущей странице
  - `/change_page` - переключение между страницами
  - `/toggle_background` - переключение фона (сохраняется для каждой страницы)
  - `/save_all_pages` - сохранение всех страниц в ZIP
  - `/update_box`, `/delete_box`, `/add_box` - работа с метками
  - `/update_position` - перемещение меток

### Frontend (частично)
- Загрузка изображений и PDF
- OCR распознавание
- Редактирование текста
- Drag & drop меток
- Белый фон у текста
- Увеличенный список меток (500px)
