## Использование

### Быстрый старт

1. Установите Poppler (см. POPPLER_INSTALL.md)
2. Запустите веб-приложение: `python web_app.py`
3. Загрузите PDF файл
4. Используйте кнопки навигации для переключения страниц
5. Запустите OCR на нужной странице

### Примеры

```bash
# Тест PDF поддержки
python scripts/test_pdf.py

# Обработать PDF
python main.py document.pdf

# Обработать страницу 5
python main.py document.pdf 5

# Пакетная обработка
python scripts/batch_pdf_ocr.py ./pdfs ./output
```

## Технические детали

### Архитектура

1. **PDFProcessor** (`utils/pdf_processor.py`)
   - Отвечает за конвертацию PDF → изображения
   - Использует `pdf2image` и Poppler
   - Поддерживает настройку DPI (по умолчанию 300)
   - Автоопределение Poppler на Windows

2. **OCRProcessor** (`main.py`)
   - Расширен методом `process_pdf()`
   - Использует PDFProcessor для конвертации
   - Применяет Tesseract OCR к изображениям
   - Поддерживает обработку всего PDF или отдельных страниц

3. **Web App** (`web_app.py`)
   - Хранит состояние PDF в `session_data`:
     - `pdf_pages` - список путей к изображениям страниц
     - `current_page` - текущая страница (0-indexed)
     - `is_pdf` - флаг PDF документа
   - Автоматически конвертирует PDF при загрузке
   - Предоставляет API для навигации

### Workflow обработки PDF

1. Пользователь загружает PDF
2. Сервер конвертирует все страницы в PNG (300 DPI)
3. Изображения сохраняются в `uploads/`
4. Первая страница отображается автоматически
5. Пользователь может:
   - Переключаться между страницами
   - Запускать OCR на текущей странице
   - Редактировать результаты
   - Сохранять обработанную страницу

## Ограничения

- PDF должны содержать растровые изображения (сканы)
- Векторный текст в PDF не поддерживается напрямую
- Большие PDF требуют много памяти
- Защищенные паролем PDF не поддерживаются
- Максимальный размер файла: 16 МБ (настраивается в web_app.py)

## Производительность

- Конвертация: ~1-2 секунды на страницу (300 DPI)
- OCR: ~2-5 секунд на страницу (зависит от содержимого)
- Память: ~50-100 МБ на страницу (300 DPI)
