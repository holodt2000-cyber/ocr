# OCR Project - Информация о проекте

## Обзор

Проект для распознавания текста с изображений с использованием Tesseract OCR.
Полностью рефакторенный, с автоматической обработкой `input.png` и комплексной документацией.

## Структура проекта

```
ocr/
├── main.py                 # Основное приложение OCR
├── test_ocr.py            # Unit тесты
├── example_usage.py       # Примеры использования
├── check_installation.py  # Проверка зависимостей
├── requirements.txt       # Python зависимости
├── .gitignore            # Игнорируемые файлы
├── LICENSE               # MIT лицензия
├── README.md             # Основная документация
├── QUICKSTART.md         # Быстрый старт
├── INSTALL.md            # Детальная установка
├── EXAMPLES.md           # Примеры кода
├── CHANGELOG.md          # История изменений
└── PROJECT_INFO.md       # Этот файл
```

## Ключевые возможности

### ✅ Реализовано

1. **Автоматическая обработка input.png**
   - Запуск без аргументов автоматически обрабатывает `input.png`
   - Поддержка явного указания пути к файлу

2. **Класс OCRProcessor**
   - Чистая архитектура с разделением ответственности
   - Настраиваемая конфигурация Tesseract
   - Обработка ошибок и валидация

3. **Логирование**
   - Информативные сообщения о процессе
   - Детальные ошибки с инструкциями по исправлению

4. **Проверка зависимостей**
   - Скрипт `check_installation.py` проверяет все компоненты
   - Понятные сообщения об отсутствующих зависимостях

5. **Комплексная документация**
   - README.md - основная документация
   - QUICKSTART.md - быстрый старт за 3 шага
   - INSTALL.md - детальная установка для всех ОС
   - EXAMPLES.md - 8+ примеров использования

6. **Unit тесты**
   - Тесты для основного функционала
   - Проверка обработки ошибок

7. **Примеры использования**
   - Базовое использование
   - Пакетная обработка
   - Предобработка изображений
   - Мультиязычное распознавание
   - Интеграция с Flask

## Технологический стек

- **Python 3.7+**
- **OpenCV** - обработка изображений
- **Pytesseract** - Python обертка для Tesseract
- **Pillow** - работа с изображениями
- **NumPy** - численные операции
- **Tesseract OCR** - движок распознавания текста

## Использование

### Быстрый старт

```bash
# 1. Проверить зависимости
python check_installation.py

# 2. Создать тестовое изображение
python example_usage.py

# 3. Обработать input.png
python main.py
```

### Программное использование

```python
from main import OCRProcessor

processor = OCRProcessor()
text = processor.process_default_input()  # Обработать input.png
print(text)
```
