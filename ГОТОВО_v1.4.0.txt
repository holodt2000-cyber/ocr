💾 ЭКОНОМИЯ ПАМЯТИ
   → Конвертируется только текущая страница

⚡ КЭШИРОВАНИЕ

   → Повторное открытие мгновенное

┌──────────────────────────────────────────────────────────────────────────────┐
│  КАК ЭТО РАБОТАЕТ                                                            │
└──────────────────────────────────────────────────────────────────────────────┘

1. ЗАГРУЗКА PDF:
   → Определяется количество страниц (быстро)
   → Конвертируется только первая страница
   → PDF открывается мгновенно

2. НАВИГАЦИЯ:
   → Нажимаете "Вперед" или "Назад"
   → Если страница в кэше → мгновенный показ
   → Если нет → конвертация (2-3 сек) → показ
   → Страница добавляется в кэш

3. ПОВТОРНЫЙ ПРОСМОТР:
   → Все просмотренные страницы в кэше
   → Мгновенное переключение

  Повтор страницы: -        →  мгновенно ✨
